H1 - HTML 1

Hvad står HTML for?: HyperText Markup Language

Hvem opfandt HTML?: Tim Berners-Lee opfandt HTML i 1990.

Hvilken version af HTML er den nyeste?: HTML5

Hvad indeholder en <body> ?: Uden en body kan du ikke lave tekst på din side.

Hvad kan <head> blandt anden indeholde og hvad SKAL det indeholde?: <head> elementet er en container for metadata og er placeret mellem <html> og <body>
