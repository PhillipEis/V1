Om CSS

Hvad er en selector med egne ord?

En selector gør så du kan definere hvilke elemnter du vil rette på, der findes både id,class og almindelig selectors

Hvad er en property med egne ord?

En Property er hvilken variable du bruger, feks color, font-weight, text-align

Hvad er en value med egne ord?

Value er selve det du propper ind, eks red,15px

Oversigt over simple selectors
Selector: Eksempel: Forklaring:
Element p Vælger alle elementer af denne type, f.eks. alle paragraphs
Id #someId Vælger den der har det id, eks id="main"
Class .answer Vælger den der har den class. Eksempel class="answer"
Universal * Kan bruges til og vælge enten alle, eller et enkelt element i din html side
Grouping table,th, td Når du "grouper" flere sammen, og giver dem den samme ting
